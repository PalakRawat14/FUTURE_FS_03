# ☕ Brew Haven Café

A modern, responsive café website built with **React.js** that delivers a premium digital experience for a fictional coffee shop. The website features an elegant user interface, interactive menu, online table reservation, customer testimonials, image gallery, and smooth animations to create an engaging and visually appealing experience across all devices.

## 💻 Getting Started

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
